# interview-assistant
AI Job Interview Assistant as part of the "Introduction to LLM" course from AI Talent Hub by 404: Answer Not Found team

## Отчет о сборе и подготовке данных для проекта
1. Процесс получения данных
Данные были собраны вручную из совокупности открытых и закрытых источников, а также специализированных статей, посвященных подготовке к собеседованиям в области Data Science (DS).

2. Содержание и структура данных
Сформированная библиотека знаний охватывает ключевые для DS темы:

Машинное обучение (ML)
Нейронные сети (Deep Learning)
Программирование на Python
SQL
Математическая статистика и алгоритмы

Основной формат данных: текстовые файлы (.txt).
Базовая единица информации: пара «Вопрос-ответ», дополненная теоретическими справками.

Хранилище: Командный репозиторий на GitHub.

Детальная структура источников:

Вопрос-ответ по темам ML из карьерной платформы Catalyst.
Книга «Введение в Data Science» от Школы анализа данных (ШАД).
Курс по LLM и агентным системам с платформы Hugging Face.
Сборник вопросов и ответов по классическому ML и Python.
3. Текущая подготовка данных
На данном этапе данные прошли первичную структуризацию по темам и источникам. Следующим шагом будет текстовый чанкинг (chunking) — разделение материалов на логические, семантически цельные фрагменты оптимального размера для дальнейшего использования языковыми моделями (например, для создания эмбеддингов или контекста RAG-систем).
